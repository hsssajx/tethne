    Tethne

    Tethne is a Python module for working with Web of Science data, currently under development by the ASU Digital Innovation Group. To start using Tethne, download and install it:

    $ git clone https://github.com/diging/tethne.git
$ cd tethne
$ python setup.py install

    You can then import Tethne as you would any other module. We recommend the following namespace conventions:

    >>> import tethne.readers as rd
>>> import tethne.networks as nt
>>> import tethne.analysis as an
>>> import tethne.writers as wr
