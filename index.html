    Tethne

    Tethne provides tools for easily parsing and analyzing bibliographic data in Python. The primary emphasis is on working with data from the ISI Web of Science database, and providing efficient methods for modeling and analyzing citation-based networks. Future versions will include support for PubMed, Scopus, and other databases.

    As of v0.3, Tethne is beginning to include methods for incorporating data from the JSTOR Data-for-Research service, and MALLET topic modeling.

    Tethne relies on NetworkX, and leverages its network analysis algorithms. You can visualize networks produced with Tethne in Cytoscape or Gephi.

    To get started, consult the tutorial.

    A demo GUI was developed for teaching purposes, and can be found in the Neotoma phenax SourceForge repository.

    Documentation

    We are developing a series of tutorials for using Tethne as a Python library, from the command-line, and via the TethneGUI.

    The latest build of the API documentation can be found here.

    Contributors

    Support or Contact

    If you run into problems, don’t panic! Tethne is under active development, and there are certainly bugs to be found. Please report any problems by opening a new issue here.
